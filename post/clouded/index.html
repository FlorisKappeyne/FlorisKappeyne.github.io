Clouded

A VR game where you redirect clouds using your hands in order to influence a simulation on the ground below you.

Clouded is the best VR game I've made so far, with very intuitive controls. I'm proud of this project because of how amazing the game turned out to be with 2 weeks development time.

Engine: Unity 5
Tools: P4V, Pivotal Tracker, Visual Studio 2017
Roles: Gameplay programmer
Team: 8 people (2 programmers, 3 designers, 3 artists)
Lifetime: IGAD Year 2 block B, 2017, 2 weeks


The project

Clouded is a VR game where the player controls the clouds. By influencing the wind in the sky, he guides clouds into forming rain or even storm clouds which influence a simulation on the ground below. The simulation is run on a hexagon grid, and each tile ranges from dry to wet (Desert, Savannah, Grass, Swamp, Water). On Grass, cities and forests spawn, which grow over time. The goal is to get as much cities as possible within a certain amount of time.

What I did

  • Everything related to the simulation on the ground, including:
    • generating a hexagon tile grid
    • letting water flow from one tile to it’s neighbours
    • randomly spawning cities or forests
    • many small game play effects and particularities
  • some small gameplay features
